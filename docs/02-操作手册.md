## 提交任务

### 步骤 1：准备 Excel 文件

**文件格式要求**：

| 列名 | 说明 | 是否必填 |
|------|------|----------|
| 品牌 | 要查询的商标名称 | ✅ 必填 |
| 国家 | 商标注册国家 | 可选 |

**示例 Excel 内容**：

| 品牌 | 国家 |
|------|------|
| Apple | US |
| Samsung | KR |
| Huawei | CN |

**注意事项**：
- ✅ 文件格式：`.xlsx` 或 `.xls`
- ✅ 第一行必须是列名
- ✅ 品牌名称不要有特殊字符
- ❌ 不要有空行

### 步骤 2：上传文件

1. 在 Dashboard 主界面，点击 **"上传文件"** 按钮
2. 选择准备好的 Excel 文件
3. 点击 **"确认上传"**

### 步骤 3：确认任务

1. 系统会预览文件内容
2. 确认品牌数量和内容
3. 点击 **"开始处理"**

### 步骤 4：等待处理

- 任务会自动加入队列
- Worker 自动开始处理
- 无需手动操作

## 查看进度

### 方法 1：Dashboard 查看

1. 访问 Dashboard：http://216.116.160.78:8080/
2. 主界面显示实时统计：
   - **总任务数**：提交的品牌总数
   - **处理中**：正在查询的品牌数
   - **已完成**：已完成查询的品牌数

3. 点击 **"刷新"** 按钮更新状态

### 方法 2：监控仪表板查看

1. 访问监控：http://216.116.160.78:8081/
2. 查看详细状态：
   - Worker 实例状态
   - 队列进度条
   - 等待中/处理中/已完成数量

### 进度状态说明

| 状态 | 含义 |
|------|------|
| ⏳ 等待中 | 任务在队列中等待处理 |
| 🔄 处理中 | Worker 正在查询该品牌 |
| ✅ 已完成 | 查询完成，结果可下载 |
| ❌ 失败 | 查询失败，需要重试 |

## 下载结果

### 步骤 1：进入结果页面

1. 在 Dashboard 点击 **"查看结果"**
2. 或点击 **"历史记录"** 查看所有批次

### 步骤 2：选择批次

1. 找到对应的任务批次
2. 确认状态为 **"已完成"**

### 步骤 3：下载文件

1. 点击 **"下载"** 按钮
2. 选择下载格式：
   - **Excel (.xlsx)**：适合查看和编辑
   - **CSV (.csv)**：适合数据导入

### 结果文件内容

| 列名 | 说明 |
|------|------|
| 品牌 | 查询的商标名称 |
| 状态 | 查询结果状态 |
| 注册号 | 商标注册号 |
| 注册日期 | 商标注册日期 |
| 申请人 | 商标申请人 |
| 类别 | 商标类别 |
| 详细信息 | 其他相关信息 |

## 查看监控

### 访问监控仪表板

地址：http://216.116.160.78:8081/

### 监控内容说明

#### 1. 系统状态

- ✅ **正常**：系统运行正常
- ⚠️ **警告**：部分功能异常
- ❌ **离线**：系统无法访问

#### 2. Worker 实例

显示 5 个 Worker 的运行状态：
- **online**：正常运行
- **offline**：已停止

#### 3. 代理池状态

- **总代理数**：253
- **可用代理**：当前可用的代理数量
- **轮换策略**：round-robin

#### 4. 队列进度

- **等待中**：待处理的任务数
- **处理中**：正在处理的任务数
- **已完成**：已完成的任务数
- **进度条**：直观显示完成百分比

#### 5. 系统资源

- **CPU 使用率**：处理器使用情况
- **内存使用率**：内存使用情况
- **磁盘使用率**：磁盘空间使用情况

### 自动刷新

- 监控仪表板每 **30 秒** 自动刷新
- 也可手动点击 **"刷新"** 按钮

## 常见操作场景

### 场景 1：批量查询 100 个品牌

```
1. 准备 Excel 文件（100 个品牌）
2. 访问 Dashboard
3. 上传文件
4. 确认并开始处理
5. 等待约 40 分钟（100 ÷ 5 × 2 分钟）
6. 下载结果文件
```

### 场景 2：查看当前任务进度

```
1. 访问监控仪表板
2. 查看队列进度部分
3. 确认处理进度
```

### 场景 3：检查系统是否正常

```
1. 访问监控仪表板
2. 查看系统状态（应为 ✅ 正常）
3. 查看 Worker 实例（应为 5 个 online）
4. 查看代理池状态（应为 ✅ 就绪）
```

---

## 注意事项

### ✅ 推荐做法

- 分批提交任务（建议每批 ≤ 200 个品牌）
- 定期查看监控仪表板
- 及时下载结果文件
- 保留原始 Excel 文件备份

### ❌ 避免做法

- 不要同时提交多个大批次任务
- 不要在处理过程中重复提交相同品牌
- 不要关闭浏览器窗口（不影响处理，但无法实时查看进度）
